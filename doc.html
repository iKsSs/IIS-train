Implementace

Skripty pro zpracování formulářů jsou umístěny v souboru function.php.
Funkce "getX", kde X jsou požadovaná data, vrací pole dat z databáze.
Správa každé kategorie je zpracována funkcí "manageX". Kategorie jsou vlaky, lístky, uživatelé/administrátoři, dopravci, stanice nebo jízdní řady.
Funkce "isX", kde X je požadované ověření, vrací logickou hodnotu (pravda/nepravda).
Lístek je možné koupi i jako neregistrovaný uživatel, v tom případě pro zadání osobních údajů je vyhrazený čas 5 minut, pak objednávka automaticky expiruje.
Automatické odhlášení je provedeno po 30ti minutové neaktivitě.
V návrhu UML bylo počítáno s externím systémem (GRAPP), který nemáme k dispozici, proto není implementována aktualizace zpoždění vlaků.

Web byl testován na standardních prohlížečích: Chrome, Firefox, Opera a IE8+ (IE8 bez stínů, jinak funkční) - na nižších verzích není zajištěna plná funkcionalita

Instalace

  • Nakopírování souborů (např. přes FTP) na server (případná úprava práv souborů/adresářů)
  • Vytvoření databáze a nakonfigurování připojení k ní v souboru config.ini
  • Import dat do databáze ze souboru vlakjecool.sql
  • Softwarové požadavky:
    • PHP verze: 5.4 a vyšší
    • DB server: InnoDB
